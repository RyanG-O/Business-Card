# 💼 Business Card Web Design

A simple **digital business card** built using HTML and CSS as part of my journey learning core computer science and front-end development fundamentals. 

---

## 📌 Overview

As a university student learning computer science, I created this project to practice foundational web development skills. The goal was to design and implement a professional-looking business card interface while reinforcing concepts like layout structure and styling.

---

## 🛠️ Built With

* HTML5 (structure and semantic layout)
* CSS3 (styling, layout, and responsiveness)

---

## ✨ Features

* Minimal and professional UI design
* Clean layout using modern CSS techniques (e.g., Flexbox)
* Organized and readable code structure
